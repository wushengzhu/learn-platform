- 安装 pnpm

```
pnpm install
```

## learn-platform-server

- 启动本地后端服务
  ```
  pnpm dev
  ```
- 打开 graphql 可视化 api 调试
  ```
  http://localhost:1024/graphql
  ```
- 生成 modules 中 api 文件模板
  ```
  pnpm temp
  ```

## learn-platform-pc

- 启动本地 pc 端
  ```
  pnpm dev
  ```
- 打开 pc 端页面
  ```
  http://localhost:1396
  ```

## learn-platform-mobile

- 启动本地 mobile 端
  ```
  pnpm dev
  ```
- 打开 mobile 端页面
  ```
  http://localhost:1398
  ```
